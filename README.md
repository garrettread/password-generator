# Password Manager Project

A RoboForm-like tkinter app built with Python.

## Features

- Random password generator
- Password strength scoring with suggestions
- Copy generated passwords to clipboard
- Save passwords to:
  - plaintext JSON
  - encrypted JSON
- Mock website login with autofill
- Credit card storage with validation
- Credit card saving to:
  - plaintext JSON
  - encrypted JSON

## To run

Clone the repo and move into the project folder:

git clone <your-repo-url>
cd password-generator

install:
pip install cryptography

run:
python main.py
